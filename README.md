# 小朋友学习任务安排小程序

一个专为6-12岁小朋友设计的学习任务跟踪管理系统，支持家长角色管理、多科目任务跟踪、进度可视化等功能。

## ✨ 功能特点

### 👨‍👩‍👧‍👦 家长角色管理
- 支持6种家长身份选择：爸爸、妈妈、爷爷、奶奶、外公、外婆
- 每个家长都可以独立布置和跟踪任务进度
- 操作记录会保存操作人信息

### 📚 8大学习任务类型
系统内置了完整的学习任务体系：

1. **拼音**
   - 摩比爱拼音
   - 学习机拼音课程
   - 目标：8月底之前全部学完

2. **古诗词**
   - 纸质书学习
   - 目标：100首

3. **文言文**
   - 纸质书学习
   - 目标：30篇

4. **数学**
   - 一年级学习机培优课程
   - 一年级奥数
   - 一年级新加坡数学
   - 目标：全部学完

5. **英文纸质书阅读**
   - 女巫温妮绘本
   - 女巫温妮桥梁书
   - 驯龙大师
   - 目标：3个月全部看完，每天看1-3本

6. **英文分级阅读**
   - 洪恩分级阅读
   - 目标：150本（6月份每周7本，7-8月每周14本）

7. **迪士尼英语课程**
   - 迪士尼英语L4和L5课程
   - 目标：完成全部400个day的课程，平均每天完成一个week内容

8. **听力训练**
   - 裸听绘本/动画片
   - 目标：每天30-60分钟

### 📊 进度跟踪功能
- 可视化进度条展示每个任务的完成情况
- 自动计算完成率和剩余进度
- 支持截止日期设置和倒计时提醒
- 统计总任务数、已完成数、完成率、今日更新任务数

### 🎨 用户体验
- 响应式设计，完美适配手机端和PC端
- 色彩丰富的界面设计，适合小朋友和家长使用
- 操作简单直观，支持点击编辑进度
- 数据自动本地存储，刷新页面不丢失

## 🚀 如何使用

1. 直接在浏览器中打开 `index.html` 文件即可使用
2. 第一次使用时选择您的家长身份
3. 点击任务卡片右上角的编辑按钮，可以更新任务进度
4. 可以使用顶部的分类标签筛选不同科目的任务
5. 所有数据会自动保存在浏览器本地

## 📱 界面预览

### 家长选择页面
- 6种家长身份可选，每个身份有不同的颜色标识
- 操作简单，一键进入系统

### 主界面
- 顶部统计卡片展示总体学习情况
- 分类标签快速切换不同科目
- 任务卡片清晰展示每个任务的进度、目标和截止时间

### 编辑页面
- 滑块快速调整任务进度
- 支持记录完成日期和学习备注
- 一键标记任务为完成

## 💾 数据存储
- 所有数据自动保存在浏览器的LocalStorage中
- 不会上传到任何服务器，数据安全有保障
- 清除浏览器数据会导致记录丢失，建议定期截图备份

## 🎯 设计理念
- 界面简洁明了，让小朋友也能看懂自己的学习进度
- 目标明确，培养小朋友的学习计划性和成就感
- 家长可以共同参与，多人协作跟踪学习情况
- 轻量化设计，无需安装，打开即用

## 🔧 技术特点
- 纯前端实现，无需后端服务器
- 使用Tailwind CSS实现响应式设计
- 本地数据持久化
- 支持所有现代浏览器

## 📝 使用建议
1. 建议家长和小朋友一起制定学习计划
2. 每天完成任务后及时更新进度
3. 可以设置小奖励，鼓励小朋友完成目标
4. 定期查看统计数据，调整学习节奏
